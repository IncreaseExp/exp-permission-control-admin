# LearnPermissionControlAdmin

> 项目正在开发...

预计接入功能为：

1. 菜单管理
2. 角色管理
3. 部门管理
4. 用户管理
5. 自定义权限分配 - 该功能较为复杂，耐心等待
6. 缓存管理

7. 服务监控
8. 日志管理

更深层次的权限开发需要看企业需求，一般的应用不需要过渡设计
